# TauREx tutorial v1 by Maël Voyer (https://mael-voyer.github.io/) 5 March 2026.
This tutorials aims to demonstrate how to run simple retrievals with TauREx. It does not covers the installation of TauREx or MultiNest and certainly does not replace reading the documentation.
This tutorial is aimed to make anyone understand the basics of TauREx. First I focus on making a forward model, then running some retrievals showcasing some of the basic capabilities of TauREx. 

Here is a list of the files inthe order you need to go through them:
- download_data.sh #Start by executing this using the following command: "bash download_data.sh". It will download the data necessary for the tutorial.
- TauREx_tutorial_1:3.pdf #First section of the tutorial
- DEMO_plot_cross_sections.ipynb #Notebook to visualize molecular cross DEMO_plot_cross_sections
- DEMO_run_and_plot_forward_model.ipynb #Notebook to run and plor a TauREx forward model.
- TauREx_tutorial_2:3.pdf #Second part of the tutorial after the first retrieval
- TauREx_tutorial_3:3.pdf #Third part of the tutorial after the second retrieval

- data/ #Empty at first but will be filled with the required data when you run "bash download_data.sh".
- data/spectra/ #Contains the different spectra you will need
